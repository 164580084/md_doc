### 目录结构

```JS{0}
|-- src
    |-- api         api请求管理文件夹
    |-- common          公共资源文件夹
    |   |-- filter          过滤器
    |   |-- font            项目字体图标、字体
    |   |-- images         公共图片存放处(静态图片、图标)
    |   |   |-- icon            公共图标存放处
    |   |   |-- index       (与pages 用到页面的文件名一样)
    |   |-- utils           公共js文件
    |-- components          通用组件
    |   |-- base            基础类组件
    |   |-- business        逻辑类组件
    |   |-- global          全局类组件
    |-- pages           页面视图

```
